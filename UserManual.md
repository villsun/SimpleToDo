Призначення програми

SimpleToDo — це графічний застосунок для керування особистими задачами. Програма дозволяє користувачу додавати, переглядати, позначати виконаними та видаляти задачі за допомогою зручного інтерфейсу.

Запуск програми

Відкрийте термінал або командний рядок.

Перейдіть у папку з програмою.

Запустіть застосунок командою:

python main.py


Після запуску відкриється графічне вікно програми.

Головне вікно програми

Головне вікно містить:

поле для введення тексту задачі;

кнопку «Додати задачу»;

кнопку «Позначити виконану»;

кнопку «Видалити задачу»;

список задач з відображенням статусу виконання.

Опис функцій
Додати задачу

Введіть текст задачі у поле вводу.

Натисніть кнопку «Додати задачу».

Задача з’явиться у списку.

Перегляд задач

Усі додані задачі відображаються у списку з нумерацією та позначкою статусу виконання.

Позначити виконану

Виділіть задачу у списку.

Натисніть кнопку «Позначити виконану».

Задача буде позначена як виконана (✓).

Видалити задачу

Виділіть задачу у списку.

Натисніть кнопку «Видалити задачу».

Задача буде видалена зі списку.

Завершення роботи

Для завершення роботи закрийте вікно програми стандартним способом.
