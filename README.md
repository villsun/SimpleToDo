SimpleToDo
Простий застосунок для управління особистими задачами.

Опис проєкту
SimpleToDo — це легкий та інтуїтивний застосунок, який дозволяє користувачам:

додавати задачі,
переглядати список задач,
позначати задачі виконаними,
видаляти непотрібні задачі

Щоб запустити тести напишіть 
cd path/to/SimpleToDo
python -m unittest discover -s unittest_todo_app


▶️ Інструкція запуску проєкту
Вимоги:

Python 3.9 або новіший

Операційна система: Windows / Linux / macOS

Кроки запуску:

Склонуйте репозиторій:

git clone https://github.com/yourusername/SimpleToDo.git


Перейдіть у папку проєкту:

cd SimpleToDo


Запустіть програму:

python main.py


Після запуску у консолі з’явиться меню керування задачами.
